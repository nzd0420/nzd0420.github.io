# Ning Z. Portfolio

A static React + Vite + TypeScript portfolio for a computer science student.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production files are generated in `dist`.

## GitHub Pages

This repository includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
Enable GitHub Pages for the repository and choose GitHub Actions as the source.
The workflow builds the site and publishes the `dist` folder.
